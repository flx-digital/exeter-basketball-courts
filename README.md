# Basketball Courts Map

A free, simple website for finding basketball courts.

## Add a court

Open `courts.json` in a text editor and copy the example entry. Change its details, including `lat` and `lng`. To find coordinates, right-click the correct spot in Google Maps and copy the first two numbers shown.

## Publish updates

In GitHub Desktop, commit your changes and click **Push origin**. GitHub Pages will normally show the update within a few minutes.

## Important

In `index.html`, change `YOUR-EMAIL@example.com` to the email address that should receive suggested updates.
